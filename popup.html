ABC WORDS

Select any text on a webpage, right-click, and choose "ABC WORDS" to see the word broken into syllables with meaning and pronunciation.

Features:

  • Breaks words into syllables
  • Shows word meaning
  • Displays pronunciation for Bengali words
  • Supports both Bengali and English text

How to use:

  1. Select any word or text on a webpage
  2. Right-click on the selected text
  3. Choose "ABC WORDS" from the context menu
  4. View the results in the popup

Important Note for Speaker Function:

If you wish to use the speaker (pronunciation playback) option, you must install Bengali and English voice packs on your computer's operating system for proper pronunciation.

How to Install Voice Packs (Windows OS):

  1. Go to your computer's Settings.
  2. Click on Time & Language.
  3. Select Language & region from the left menu.
  4. Under "Preferred languages", click on Add a language.
  5. Search for and select Bengali (India) or Bengali (Bangladesh), and English (United States) or your preferred English variant.
  6. After selecting, click Next. On the next screen, ensure the Speech or Text-to-speech checkbox is selected for the chosen language. This will download and install the voice pack.
  7. Click Install and wait for the process to complete.

How to Install Voice Packs (Linux OS):

On most Linux distributions, you can install text-to-speech packages and language data using your system's package manager. Common text-to-speech engines include `espeak` or `festival`.

  1. Open Terminal: Use `Ctrl+Alt+T` or search for "Terminal".
  2. Update package lists:
    sudo apt update (for Debian/Ubuntu based systems)
  3. Install text-to-speech engine (e.g., espeak and its data):
    sudo apt install espeak espeak-ng-data mbrola-bn1 (for Bengali voice, if available)
  4. Install English voice:
    sudo apt install espeak-ng-data-en (or similar package)
  5. Verify installation (optional):
    espeak -v bn "এটি একটি পরীক্ষা"
    espeak -v en "This is a test"
  6. For Fedora/RHEL based systems: Use `sudo dnf install espeak espeak-ng-data` and search for specific language packs.

How to Install Voice Packs (macOS):

macOS usually has robust built-in text-to-speech capabilities.

  1. Go to System Settings (or System Preferences on older macOS versions).
  2. Click on Accessibility.
  3. In the sidebar, click on Spoken Content.
  4. Next to "System Voice", click on Manage Voices....
  5. Find and select the desired Bengali and English voices. You may need to scroll down or search. Check the box next to them to download.
  6. Once downloaded, they will be available for use.

After installation, reload your extension in Chrome (go to chrome://extensions/ and click the refresh button for ABC WORDS) for the changes to take effect.
